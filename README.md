A simple mod that adds some variant colors to the crude torch item type.

You can craft these torches the same way you can regular crude torches, via a workbench or pocket crafting. The ingame items do not have different names or require other materials to craft; you can tell them apart by their icon color.

**Adds**

- Blue Crude Torch
- Cyan Crude Torch
- Green Crude Torch
- Magenta Crude Torch
- Red Crude Torch
- White Crude Torch
